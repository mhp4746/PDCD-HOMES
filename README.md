# PDCD Homes — Construction Tracker

A real-time site progression tracker for residential home building. Built for directors and supervisors to manage multiple active construction projects from one dashboard.

## Features

- **Director Dashboard** — See all active projects at a glance with progress bars, current stage, and budget
- **Task Tracking** — 100+ tasks across 8 construction stages (Preliminary → Handover), based on real construction workflows
- **Task Notes** — Supervisors can add timestamped notes to any task (e.g. "Plumber delayed, rescheduled to Thursday")
- **Project Notes** — General project-level notes for client updates, council approvals, etc.
- **Add/Remove Projects** — Director creates new projects with address, client, dates, and budget
- **Milestone Claims** — Payment claim tasks highlighted so nothing gets missed
- **Zero Training** — Supervisors just tick checkboxes. That's it.

## Tech Stack

- React 18
- Vite
- Deployed on Vercel (free tier)

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Push to GitHub → Connect to Vercel → Auto-deploys on every push.
